# ADUFlow

ADUFlow is a builder-focused sales platform for modular construction companies.

## Core idea

A modular builder uploads standard models, pricing, options, and BOM data. Customers configure a modular building online and receive an instant budget, proposal, and next-step process.

## MVP focus

Version 1 focuses on:

- Modular ADU / garden-suite configurator
- Live pricing engine
- Proposal generation
- Lead capture
- Basic BOM output
- Builder dashboard for models and options

## Target users

- ADU builders
- Garden-suite builders
- Tiny-home builders
- Modular-home builders
- Workforce-housing builders
- Modular garage and cabin builders

## Tech stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
- PDF proposal generation

## Development phases

1. Landing page and validation funnel
2. Customer configurator
3. Pricing engine
4. Proposal generator
5. Builder dashboard
6. BOM engine
7. Builder upload wizard
8. Permit and project workflow modules

## Positioning

ADUFlow is not another marketplace. It is a sales operating system for modular builders.

Primary promise:

> Reduce quote time from days to minutes.
