# ADUflow Pilot Presentation

## Slide 1 - Title

ADUflow turns early ADU chaos into a bankable project workflow.

Address lookup, zoning screen, builder catalog pricing, proposal, lender package, permit checklist, and project tracker.

## Slide 2 - The Problem

ADU builders lose time before the project is real.

- Homeowners do not know if the lot works
- Builders manually explain zoning and setbacks
- Financing needs a stronger package than a rough quote
- Permit tasks and draw milestones live in scattered spreadsheets

## Slide 3 - The Wedge

Feasibility before configuration.

The homeowner starts with a property address. ADUflow returns a first-pass zoning screen and then lets the homeowner select eligible models and options.

## Slide 4 - What The Builder Gets

- Lead pipeline
- Builder-managed model catalog
- Builder-managed option pricing
- Proposal packages
- Lender package export
- Permit and HOA checklist
- Project milestone tracker
- Basic analytics

## Slide 5 - Address-First Zoning

ADUflow captures:

- zoning source
- zone classification
- max ADU size
- max stories
- setbacks
- ADU permitted flag
- review risk
- permit path

When no live provider key is present, the demo uses a clearly labeled municipal fallback estimate.

## Slide 6 - Proposal Package

Every intake generates a proposal with:

- homeowner and property
- model and square footage
- feasibility result
- zoning source
- budget range
- factory/site cost split
- timeline
- next steps
- share link and PDF print flow

## Slide 7 - Lender Package

The lender package turns the quote into a financing document:

- property and project overview
- zoning and feasibility snapshot
- factory/site cost summary
- draw release schedule
- evidence requirements
- contingency/disclaimer language

## Slide 8 - Permit Assistant

The permit assistant creates:

- property intake tasks
- zoning review tasks
- drawing package checklist
- engineering checklist
- utilities and site checklist
- municipal submittal tasks
- HOA package when relevant

## Slide 9 - Project Lifecycle

After proposal, ADUflow tracks:

- contract signed
- permit submitted
- permit approved
- factory production start
- foundation ready
- factory completion
- delivery and set
- weather-tight
- final inspection
- occupancy

## Slide 10 - Pilot Ask

We are looking for 3-5 builders to pilot ADUflow.

Builder provides:

- model list
- base pricing
- option pricing
- service region
- builder credentials
- feedback from real leads

ADUflow provides:

- live configurator
- generated proposals
- lender package
- permit checklist
- dashboard

## Close

The goal is simple:

Help builders decide whether a homeowner lead is feasible, financeable, and worth pursuing before anyone spends hours on manual scoping.
