The ADUflow flow

One homeowner intake creates the operating package.

1
Address-first zoning screen

The homeowner enters an address. ADUflow returns the zoning source, zone, max ADU size, setbacks, review risk, and permit path when available.

2
Builder catalog pricing

The configurator uses the builder's models and options, then calculates factory cost, site cost, range, and timeline.

3
Proposal and share link

Every intake creates a proposal package with feasibility, cost split, model summary, and dynamic next steps.

4
Lender and permit outputs

The same lead generates a printable lender package, permit checklist, and project tracker.

Address lookup

Feasibility starts with the property, not the floor plan.

With a live zoning API key, ADUflow can call the provider. Without one, it uses clearly labeled municipal fallback rules for demo markets.

What the screen returns

  • Zoning classification and description
  • Maximum ADU size and stories
  • Front, side, and rear setback guidance
  • ADU permitted flag when available
  • Review risk and permit path

Important label

  • Zoneomics or other provider: provider result
  • Fallback data: municipal fallback estimate
  • Manual scenario: builder-selected assumption
  • No result is a permit approval

Builder setup

Your catalog and credentials drive the outputs.

Models

Add building names, codes, square footage, base prices, regions, and active status.

Options

Manage finish, foundation, utilities, and site options with stable values, descriptions, and prices.

Credentials

Store builder legal name, license, insurance, warranty, and service region for lender exports.

Analytics

Track proposal value, lead status, likely feasible leads, high-risk leads, and model demand.

Lender package

The financing output is a printable package, not a loose estimate.

Included sections

  • Homeowner and property address
  • Project scope and model summary
  • Zoning source and feasibility snapshot
  • Factory/site cost breakdown
  • Budget range and contingency note

Draw schedule

  • Deposit and permit package
  • Foundation ready
  • Factory completion
