Getting set up

Load your models and pricing options.

Go to Builder OS → Manage catalog. This is where you load your real models so the configurator shows your actual lineup instead of the demo defaults.

Models tab

  • Add each building model you offer
  • Set the name, square footage, and base factory price
  • You can edit or deactivate models any time

Options tab

  • Four categories: Finish, Foundation, Utilities, Site
  • Add the option names, descriptions, and prices you charge
  • These become the choices in the configurator

💡 If you're not sure about exact option prices yet, start with ballpark figures. You can refine them any time — the configurator updates immediately.

The configurator

What a homeowner experiences.

1
Pick a parcel scenario

Urban lot with lane, deep suburban, corner/HOA, or tight services. This sets the feasibility context — max size, setback, permit path, review risk.

2
Select a model and options

Building model, finish level, foundation type, utilities scope, site condition. Every choice updates the estimate instantly — no page reload.

3
Review the estimate panel

Feasibility result and confidence score, total budget range, factory vs. site cost split, permit path, draw milestones. All visible before they submit anything.

4
Submit the intake form

Name, email, phone, property address. This creates the lead in your dashboard and generates the proposal package.

Proposals

Every lead gets a full proposal — automatically.

The proposal package is generated the moment a homeowner submits. You don't write it — it's built from their configuration.

What's in it

  • Project snapshot — homeowner, address, proposal number
  • Feasibility result and confidence score
  • Full budget with factory / site split
  • Design envelope — model, max sqft, setbacks, timeline
  • Dynamic next steps based on permit path and risk level

What you can do with it

  • Open from your Builder OS dashboard
  • Copy a share link — send directly to the homeowner
  • Download as PDF — print or attach to an email
  • Generate the permit checklist with one click

Permit assistant

Turn every proposal into a permit action plan.

From the proposal page, click Generate permit checklist. ADUflow builds a task list and document checklist automatically — adapted to the lead's parcel scenario and permit path.

Task categories

  • Property intake and survey confirmation
  • Zoning and setback validation
  • Drawing package requirements
  • Engineering sign-off
  • Utilities and site review
  • Municipal submittal and HOA package (if applicable)

Document checklist

  • Survey or site plan base
  • Architectural drawings (site plan, floor plan, elevations)
  • Foundation and engineering plan
  • Utility tie-in plan
  • Preliminary budget and draw schedule (for lender)
  • HOA exterior package (if corner lot or design review)
